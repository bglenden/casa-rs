Run the 2x2 cross-matrix using execute_taql() on the Rust side.

Checks all 4 cells execute without error, but does NOT compare result grids. Use this for queries that produce computed columns (GROUP BY, expressions in SELECT, array indexing in SELECT, etc.) where table.query() returns something other than a RefTable.
