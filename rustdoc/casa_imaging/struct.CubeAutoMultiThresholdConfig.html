Expand description

CASA auto-multithresh controls used by the cube CLEAN controller.

Fields§

§sidelobe_threshold: f32

Sidelobe threshold factor multiplied by the PSF sidelobe level.

§noise_threshold: f32

Noise threshold factor multiplied by the robust residual RMS.

§low_noise_threshold: f32

Lower noise threshold factor used when growing an existing mask.

§negative_threshold: f32

Negative-feature threshold factor; zero disables negative masks.

§smooth_factor: f32

Smoothing factor, in restoring-beam units, for CASA’s mask smoothing stage.

§min_beam_frac: f32

Minimum connected-region size as a fraction of the fitted beam area.

§cut_threshold: f32

Fraction of the smoothed-mask peak used to cut mask edges.

§grow_iterations: usize

Maximum constrained binary-dilation iterations for later mask updates.

§do_grow_prune: bool

Whether grown masks are pruned after dilation.

§min_percent_change: f32

CASA percent-change stop control for later automask updates.

§fast_noise: bool

Use CASA’s fast-noise statistics path.

pub fn build_cube_clean_mask( &self, geometry: ImageGeometry, dirty: &CubeImagingResult, mask_beams: &[Option<BeamFit>], user_mask: Option<&Array2<bool>>, ) -> Result<Array4<bool>, ImagingError>

Builds a CASA-style auto-multithresh clean-mask cube from residual products and per-plane diagnostics.

The returned mask has shape [nx, ny, 1, nchan]; multiple residual Stokes planes are OR-combined into the single persisted clean-mask plane for each channel. mask_beams should contain the restoring beam selected for each output channel after any common-beam policy has been applied.
