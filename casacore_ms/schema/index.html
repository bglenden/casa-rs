Expand description

MS table and subtable schema definitions.

Each submodule exports REQUIRED_COLUMNS and OPTIONAL_COLUMNS arrays of ColumnDef constants. The main_table module additionally exports keyword definitions.

The SubtableId enum identifies the 17 MS subtables (12 required, 5 optional) and provides name() / is_required() accessors.
