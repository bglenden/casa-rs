Expand description

Pixel types supported by the generic tile I/O path.

Implemented for bool, f32, f64, Complex32, and Complex64. ELEM_SIZE is the unpacked in-memory element size used by the tile cache; SWAP_SIZE is the component size for byte-swapping (1 for bool, 4 for f32/Complex32, 8 for f64/Complex64).
