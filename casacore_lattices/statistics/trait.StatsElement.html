Expand description

Marker trait for lattice element types that support numerical statistics.

Implemented for all numeric casacore-native types (bool, integers, floats). Not implemented for String, num_complex::Complex32, or num_complex::Complex64; those require separate handling.

Mirrors C++ NumericTraits<T>::PrecisionType by providing a conversion to f64 for accumulation.
