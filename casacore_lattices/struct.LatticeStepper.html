Struct LatticeStepper

pub struct LatticeStepper { /* private fields */ }
Expand description

A navigator that steps through a lattice in rectangular chunks along a configurable axis path.

Corresponds to the C++ LatticeStepper class. The cursor advances sequentially through the lattice, with axis ordering determined by the axis_path. By default, the fastest-varying axis (axis 0) is iterated first.

Near lattice boundaries, the cursor shape is reduced (hangover handling) so that no out-of-bounds access occurs.

§Examples

use casacore_lattices::{LatticeStepper, LatticeNavigator};

let stepper = LatticeStepper::new(vec![10, 10], vec![5, 5], None);
assert_eq!(stepper.n_steps(), 4); // 2×2 steps

pub fn new( lattice_shape: Vec<usize>, cursor_shape: Vec<usize>, axis_path: Option<Vec<usize>>, ) -> Self

Creates a new LatticeStepper.

  • lattice_shape: the overall lattice dimensions.
  • cursor_shape: the desired chunk size along each axis.
  • axis_path: optional axis ordering (default: 0, 1, 2, …). The first axis in the path is iterated fastest.
