Expand description

Pixel types supported by the generic tile I/O path.

Implemented for f32, f64, Complex32, and Complex64. ELEM_SIZE is the on-disk element size; SWAP_SIZE is the component size for byte-swapping (4 for f32/Complex32, 8 for f64/Complex64).
