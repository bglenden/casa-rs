Typed wrappers for MS subtables.

Each subtable wrapper holds a reference to the underlying [Table] and provides typed accessor methods for the columns defined in the corresponding schema module.

Immutable wrappers (Ms*<'a>) provide read access; mutable wrappers (Ms*Mut<'a>) additionally support write operations.
