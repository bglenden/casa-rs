Quantum column support: unit-tagged table columns.

This module provides the Rust counterparts of C++ TableQuantumDesc, ScalarQuantColumn, and ArrayQuantColumn from the casacore measures/TableMeasures subsystem.

A “quantum column” is a regular numeric column whose values carry physical units. The unit metadata is stored in the column’s keyword record as either:

  • QuantumUnits — a Vector<String> of fixed unit name(s), or
  • VariableUnits — the name of a companion String column that holds per-row (scalar) or per-element (array) unit strings.

§Examples

use casacore_tables::{Table, TableSchema, ColumnSchema, table_quantum::TableQuantumDesc};
use casacore_types::*;

// Create a table with a quantum column
let schema = TableSchema::new(vec![
    ColumnSchema::scalar("flux", PrimitiveType::Float64),
]).unwrap();
let mut table = Table::with_schema(schema);

// Attach fixed units
let desc = TableQuantumDesc::with_unit("flux", "Jy");
desc.write(&mut table).unwrap();
assert!(TableQuantumDesc::has_quanta(&table, "flux"));
