Shared typed parameters, sparse TOML profiles, and managed Last state.

This crate owns parameter lifecycle rather than provider semantics. The latter live in [casa_provider_contracts], allowing CLI, TUI, GUI/UniFFI, Python, and providers to resolve the same self-contained surface bundle.
