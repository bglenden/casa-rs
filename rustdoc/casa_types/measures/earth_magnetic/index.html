Earth-magnetic measure support.

This module provides:

  • EarthMagneticRef — Earth-magnetic reference frame types matching the C++ MEarthMagnetic::Types catalog.
  • MEarthMagnetic — a magnetic-field vector in nanotesla tagged with a reference frame, equivalent to C++ casa::MEarthMagnetic.
  • calculate_igrf — a reusable IGRF model helper corresponding to the C++ EarthMagneticMachine path used by TaQL meas.igrf* UDFs.

The explicit vector conversions reuse the same frame-routing machinery as super::direction::MDirection: the vector direction is converted between reference frames while the field strength is preserved.
