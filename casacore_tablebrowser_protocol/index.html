Wire protocol for the tablebrowser subprocess session.

The protocol is versioned JSON Lines over stdio. Every client request and server response is a single JSON object terminated by \n.

Version 1 is intentionally read-only, but it already carries stable value addresses, typed inspector payloads, and capability flags so later edit-mode extensions can target the same contract without a wire redesign.
