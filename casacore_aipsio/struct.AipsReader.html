Expand description

Streaming reader for casacore primitive values in canonical wire format.

AipsReader is the read-side counterpart to AipsWriter. It decodes individual scalars and arrays without any object-framing interpretation; callers that need type-checked, versioned object persistence should use aipsio::AipsIo instead.

The byte order must match the byte order used by the writer that produced the stream. Use ByteOrder::BigEndian (the default) for files written by standard casacore tools.

Read a boolean value from a single byte (0 → false, 1 → true).
