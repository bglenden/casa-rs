2//! Built-in unit definitions ported from C++ casacore `UnitMap2.cc`–`UnitMap7.cc`.
3//!
4//! Each table is a `const` slice of `(name, factor, description, dims)` tuples
5//! where `dims` is the `[i8; 10]` SI dimension exponent array.
6//!
7//! The factor is always the SI-coherent conversion factor.  For derived units
8//! whose C++ definition references other units (e.g. `"J" = 1 "N.m"`), the
9//! factor stored here is the fully-resolved SI value.
10//!
11//! # Dimension index order
12//!
13//! `[m, kg, s, A, K, cd, mol, rad, sr, _]`
14
