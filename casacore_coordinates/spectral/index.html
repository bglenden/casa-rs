One-axis spectral (frequency/velocity) coordinate.

SpectralCoordinate implements a linear mapping between pixel position and frequency (or velocity) in a specified reference frame. This corresponds to C++ SpectralCoordinate in its simplest (non-tabular) mode.

The conversion is:

world = crval + cdelt * (pixel - crpix)
