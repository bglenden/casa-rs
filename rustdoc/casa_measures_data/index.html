CASA-table-backed measures runtime data for astronomical conversions.

The runtime model follows CASA’s layout and defaults:

  • default measures path: ~/.casa/data
  • preferred source: an existing CASA/casaconfig-populated tree
  • fallback for the default path only: a packaged CASA-table snapshot shipped with casa-rs and unpacked into ~/.casa/data on first use
  • explicit override: CASA_RS_MEASURESPATH (with deprecated compatibility alias CASA_RS_DATA)
