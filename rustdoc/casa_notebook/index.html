Rust-owned scientific notebook and execution-receipt contracts.

The visible source of truth is ordinary Markdown. Managed execution receipts live separately under .casa-rs/notebook-runs/. This crate owns both persisted formats so native, terminal, command-line, and Python surfaces cannot drift into independent storage implementations.
