Struct TiledFileIO

Expand description

Random-access tile I/O for a TiledCellStMan data file.

Provides direct read/write of individual tiles without loading the entire array into memory. Uses an in-memory tile cache (modeled on C++ casacore’s BucketCache) to avoid redundant disk I/O. Tiles are byte-swapped on cache load/flush, so the hot path operates on native-endian data.

§C++ equivalent

TSMCube + BucketCache in casacore’s tiled storage manager.
