# Branding Assets

Truth class: current descriptive
Last reality check: 2026-05-09

This directory holds generated visual identity assets for `casa-rs`.

## Files

- `headers/casa-rs-header.png`: wide documentation and website header image.
- `app-icon/casa-rs-app-icon.icns`: macOS app icon bundle.
- `app-icon/casa-rs-app-icon-1024.png`: 1024px square icon source.
- `app-icon/png/`: direct PNG icon exports from 16px through 1024px.
- `source/`: original generated source PNGs for future revisions.

## Generation Notes

The assets are based on the first selected concept: a radio-observatory AI
assistant with side dish panels, a copper Rust-inspired gear halo, cyan
interferometry arcs, and dark observatory-night radio-array imagery. They are
intended to nod to CASA's radio astronomy heritage while remaining visually
distinct from official CASA branding.
