Recursive descent parser for unit expressions.

Parses strings like "km/s", "kg.m.s-2", "Jy/beam" into a UnitVal. The grammar matches the C++ UnitVal::create() parser.

§Grammar

unit_expr = term { ('.' | '/') term }*
term      = '(' unit_expr ')' [exponent]
          | field [exponent]
field     = known_unit                   # try full name
          | prefix(1) known_unit         # 1-char prefix + rest
          | prefix(2) known_unit         # 2-char prefix + rest
exponent  = ['-'] digits

The separator between terms is . (multiply) or / (divide). Terms without a separator are multiplied.
