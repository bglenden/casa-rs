pub fn rust_taql_exec_ok(table: &mut Table, query: &str) -> Result<(), String>
Expand description

Execute a TaQL statement via execute_taql(), returning Ok(()) on success.

Use this for queries that produce computed columns (GROUP BY, expressions in SELECT, etc.) where table.query() would fail because the result is not a simple RefTable.
