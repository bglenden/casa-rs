Enum MeasureType 

Source 
pub enum MeasureType {
    Epoch,
    Direction,
    Position,
    Frequency,
    Doppler,
    RadialVelocity,
}
Expand description

The kind of measure stored in a column.

Corresponds to the type field in the MEASINFO sub-record.
