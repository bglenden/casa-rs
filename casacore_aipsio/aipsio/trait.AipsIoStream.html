Expand description

Object-safe byte stream used as the backing store for AipsIo.

Any type implementing Read + Write + Seek + Any automatically satisfies this trait. In practice the two common implementors are std::io::Cursor<Vec<u8>> (in-memory) and std::fs::File (file-backed). The into_any method enables downcasting back to the concrete type via AipsIo::into_inner_typed.
