2//! Astronomical measures: typed physical quantities with reference frames.
3//!
4//! This module provides the casacore *measures* system — physical quantities
5//! (epochs, positions, directions, frequencies, Doppler shifts) that carry
6//! both a numeric value and a reference frame. Reference-frame conversions
7//! use the IAU SOFA algorithms via the [`sofars`] crate.
8//!
9//! ## Key types
10//!
11//! - [`MEpoch`] — a time instant in one of 12 time scales (UTC, TAI, TT, etc.),
12//!   equivalent to C++ `casa::MEpoch`.
13//! - [`MPosition`] — a 3D position in ITRF or WGS84, equivalent to C++
14//!   `casa::MPosition`.
15//! - [`MDirection`](direction::MDirection) — a sky direction in J2000, GALACTIC,
16//!   etc., equivalent to C++ `casa::MDirection`.
17//! - [`MEarthMagnetic`](earth_magnetic::MEarthMagnetic) — an Earth magnetic
18//!   field vector tagged with a reference frame, equivalent to C++
19//!   `casa::MEarthMagnetic`.
20//! - [`MFrequency`](frequency::MFrequency) — a spectral frequency in LSRK,
21//!   BARY, TOPO, etc., equivalent to C++ `casa::MFrequency`.
22//! - [`MDoppler`](doppler::MDoppler) — a Doppler shift in RADIO, Z, BETA, etc.,
23//!   equivalent to C++ `casa::MDoppler`.
24//! - [`MeasFrame`] — conversion context carrying epoch, position, direction,
25//!   and UT1−UTC offset.
26//! - [`MjdHighPrec`] — high-precision Modified Julian Date (day + fraction),
27//!   equivalent to C++ `MVEpoch`.
28//!
29//! ## Record serialization
30//!
31//! The [`record`] submodule provides `to_record` / `from_record` functions
32//! that produce [`RecordValue`](crate::RecordValue) in the same layout as C++
33//! `MeasureHolder`, ensuring on-disk compatibility.
34//!
35//! ## Examples
36//!
37//! ```no_run
38//! use std::sync::Arc;
39//! use casa_types::measures::{MEpoch, EpochRef, MeasFrame, MeasuresProvider};
40//!
41//! # fn application_measures_provider() -> Arc<dyn MeasuresProvider> {
42//! #     todo!("open and inject the application's measures runtime")
43//! # }
44//!
45//! // Convert J2000.0 from UTC to TAI
46//! let utc = MEpoch::from_mjd(51544.5, EpochRef::UTC);
47//! let frame = MeasFrame::new().with_measures(application_measures_provider());
48//! let tai = utc.convert_to(EpochRef::TAI, &frame).unwrap();
49//! let diff_s = (tai.value().as_mjd() - utc.value().as_mjd()) * 86400.0;
50//! assert!((diff_s - 32.0).abs() < 0.01);
51//! ```
