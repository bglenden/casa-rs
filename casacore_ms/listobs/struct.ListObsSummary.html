Expand description

Structured listobs summary for a MeasurementSet.

schema_version is included so JSON consumers can pin their deserializers to a known shape as this summary grows. New optional fields may be added in future schema versions without changing the default text output.

Fields§

§schema_version: u32

Version of the JSON schema emitted by render_json_pretty.

§options: ListObsOptions

Options applied while generating this summary.

§measurement_set: MeasurementSetInfo

General information about the MeasurementSet being summarized.

§observations: Vec<ObservationSummary>

OBSERVATION subtable rows.

§scans: Vec<ScanSummary>

Aggregated MAIN-table scan summaries, grouped by observation, array, and scan number.

§fields: Vec<FieldSummary>

FIELD subtable rows.

POLARIZATION subtable rows.

§data_descriptions: Vec<DataDescriptionSummary>

DATA_DESCRIPTION subtable rows.

§spectral_windows: Vec<SpectralWindowSummary>

SPECTRAL_WINDOW subtable rows with derived polarization information.

§sources: Vec<SourceSummary>

SOURCE subtable rows.

§antennas: Vec<AntennaSummary>

ANTENNA subtable rows.
