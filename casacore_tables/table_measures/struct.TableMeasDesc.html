Struct TableMeasDesc

pub struct TableMeasDesc { /* private fields */ }
Expand description

Descriptor for a measure column’s MEASINFO metadata.

Corresponds to C++ casa::TableMeasDesc. This descriptor records the measure type, reference frame mode (fixed, variable-int, variable-string), and optional offset for a table column.
