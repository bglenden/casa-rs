Struct TempImage 

Source 
pub struct TempImage<T: ImagePixel> { /* private fields */ }
Expand description

A temporary image that stores pixels in a [TempLattice] and metadata in member fields.

Corresponds to C++ casacore::TempImage<T>. Unlike PagedImage, a TempImage never writes to a casacore table unless explicitly materialized via save_as or into_paged.

Small images (below the default 256 Ki element threshold) live entirely in memory. Larger ones use a scratch PagedArray that auto-manages its own temporary file.
