Expression evaluator for TaQL.

Evaluates Expr nodes against table row data, producing ExprValue results with automatic type promotion.

§Type promotion hierarchy

Matches C++ TaQL: Bool → Int → Float → Complex. All integer types widen to i64, all floats to f64.
