Physical constants as Quantity values.

This module provides the ~20 fundamental constants from C++ casa::QC, each as a lazily-initialised &'static Quantity.

§Examples

use casacore_types::quanta::constants;

let c = constants::c();
assert!((c.value() - 2.99792458e8).abs() < 1.0);
assert_eq!(c.unit().name(), "m/s");
