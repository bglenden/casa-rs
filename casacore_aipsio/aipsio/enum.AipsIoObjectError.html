Expand description

Errors from AipsIo object-framing operations.

These cover three broad categories:

  • Framing / state — calling put/get methods in the wrong order or when the stream is not open.
  • Type / format — magic values missing, type-string mismatch on AipsIo::getstart, unrecognised wire tags.
  • Bounds / shape — reading past the end of an object, or array shapes that cannot be represented on the current platform.

I/O and UTF-8 errors from the underlying stream are wrapped and propagated transparently.

§

Io(Error)

A low-level I/O error from the underlying stream.

§

Utf8(FromUtf8Error)

The stream contained a byte sequence that is not valid UTF-8 when decoding a string field.

§

PutStartUnavailable

AipsIo::putstart was called when the stream is not open or is read-only.

§

GetNextTypeUnavailable

AipsIo::get_next_type was called when the stream is not open or is write-only.

§

NoPutStart

A scalar or array put method was called outside a putstart/putend block.

§

NoGetStart

A scalar or array get method was called outside a getstart/getend block.

§

ReadBeyondEndOfObject

Reading would exceed the declared length of the current object.

§

NoMagicValueFound

AipsIo::get_next_type did not find the expected top-level magic value (0xbebebebe) at the current stream position.

§

ObjectTypeMismatch

AipsIo::getstart found a different object type than the caller expected.

§found: String
§expected: String
§

IncompleteObjectRead

AipsIo::getend detected that not all bytes of the object were consumed.

§

SetPosWhileAccessingObjects

AipsIo::setpos was called while inside an object (nesting level > 0).

§

AlreadyOpen

AipsIo::open_file was called on an instance that already has an open stream.

§

NotOpen

A method requiring an open stream was called after the stream was closed.

§

StreamTypeMismatch

AipsIo::into_inner_typed was called with a concrete type that does not match the actual underlying stream type.

Fields

§expected: &'static str
§

LengthOverflow

An object-length or count value overflowed a u32.

§

InvalidValueKindTag(u8)

The wire-encoded value-kind tag byte did not match any known variant.

§

UnsupportedValueKind(ValueKind)

AipsIo::put_value was called with a Value variant this object codec does not support.

§

InvalidPrimitiveTypeTag(u8)

The wire-encoded primitive-type tag byte did not match any known variant.

§

DimensionOverflow(u64)

An array dimension read from the stream cannot be converted to usize on this platform.

§

InvalidArrayShape

The product of the array shape dimensions does not match the number of elements in the payload.
