Reusable CASA-style selector parsing for MeasurementSet-facing APIs.

CASA keeps generic selector parsing near MSSelection, while spectral cube regridding and axis construction live lower in SubMS / MSTransformRegridder. This module covers the former: reusable text parsing for numeric-id selectors and SPW+channel selectors.
