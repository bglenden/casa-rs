Expand description

Replayable source of bounded standard-MFS planned sample blocks.

This is the core streaming input boundary for standard-MFS backends. A frontend may read MeasurementSet rows, apply weighting, and plan compact tap identities however it chooses, but it must be able to replay the same bounded blocks for the initial dirty/PSF pass and every exact residual refresh pass.
