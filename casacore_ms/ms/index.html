High-level MeasurementSet type that manages main table + subtables as a unit.

A MeasurementSet bundles the MS main table with its required (and optional) subtables. It can be created from scratch via MeasurementSet::create, opened from disk via MeasurementSet::open, or assembled in memory for testing.

Cf. C++ MeasurementSet class.
