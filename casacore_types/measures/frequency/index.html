Frequency measure: spectral frequencies in various velocity reference frames.

This module provides:

  • FrequencyRef — velocity reference frame types (LSRK, BARY, TOPO, etc.).
  • MFrequency — a frequency in a specified reference frame, equivalent to C++ MFrequency.

Conversions between frames use relativistic Doppler shifts computed from the radial velocity of one frame relative to another along the line of sight.

§Velocity constants

The standard velocity vectors (from casacore MeasTable) are defined in J2000 coordinates as direction × speed (m/s):
