Array-level mathematical utilities mirroring C++ casacore/casa/Arrays/ArrayMath.h.

ndarray already covers element-wise arithmetic and basic reductions (sum, mean). The functions here fill the gaps needed for idiomatic casa-rs code:

RustC++ equivalent
array_mediancasacore::median(arr)
array_fractilecasacore::fractile(arr, fraction)
nearcasacore::near(a, b) (f64 default tol=1e-13)
near_f32casacore::near(a, b) (f32 default tol=1e-5)
near_tolcasacore::near(a, b, tol)
near_abscasacore::nearAbs(a, b, tol)
