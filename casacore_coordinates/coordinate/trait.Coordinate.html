Expand description

The interface shared by all coordinate types.

Every concrete coordinate (e.g. LinearCoordinate, DirectionCoordinate) implements this trait. The trait is object-safe (via clone_box) so that coordinates can be stored as Box<dyn Coordinate> inside a CoordinateSystem.

The pixel-to-world and world-to-pixel methods follow the conventions of C++ Coordinate::toWorld and Coordinate::toPixel.
