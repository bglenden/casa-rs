Export real standard-MFS weighted samples for the local Metal experiment harness.

This is a measurement aid, not a production backend path. It streams the MeasurementSet through the same standard-MFS density and routed visibility paths as the fixed-tile frontend, then writes only the compact per-lane fields consumed by tools/experiments/metal.
