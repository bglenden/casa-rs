Per-row derived column accessors for the MS main table.

DerivedColumns wraps an MsCalEngine and provides per-row accessors that read ANTENNA1, ANTENNA2, TIME, and FIELD_ID from the main table and delegate to the engine for computation.

Cf. C++ DerivedColumn.
